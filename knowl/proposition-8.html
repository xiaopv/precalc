Task 3.1.

If we start with a single yeast cell under favorable growth conditions, then it will divide in one hour to form two identical "daughter cells". In turn, after another hour, each of these daughter cells will divide to produce two identical cells; we now have four identical "granddaughter cells" of the original parent cell. Under ideal conditions, this "doubling effect" will continue.
  1. Create a table that contains the number of yeast cells \(N(t)\) at any given hours between \(t=0\) and \(t=8\text{.}\)
  2. How many yeast cells are there when \(t=100\text{?}\) (If you are stuck with this question or it takes too long, move on to the next question and then come back to this one later.)
  3. Write a function equation of the number of yeast cells \(N(t)\) in terms of the hours \(t\text{.}\)
  4. If you were not able to answer Question 2, can you answer that question now using the function equation from Question 3?
