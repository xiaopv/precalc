Task 3.7.

Let us re-consider the yeast cell growth problem. This time, we will start with three yeast cells instead of one.
  1. Create a table that contains the number of yeast cells \(M(t)\) at any given hours between \(t=0\) and \(t=8\text{.}\)
  2. Can you use an exponential function to model this yeast cell growth problem?
  3. Now model this problem using a function of exponential type. Write a function equation of the number of yeast cells \(M(t)\) in terms of the hours \(t\text{.}\)
  4. What is the domain and the range of the function \(M(t)\text{.}\)
  5. After how many hours will the number of yeast cells be exactly \(98304\text{?}\)
