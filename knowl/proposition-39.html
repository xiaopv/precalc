Task 5.5.

Given any function \(f(x)\text{,}\) describe how the graph of each function below will differ from the graph of \(f(x)\text{.}\) Try not to use any specific function equations to answer this question but if you are stuck, take \(f(x)=x^2\) and experiment it using Desmos.
  1. \(\displaystyle 3f(x-2)\)
  2. \(\displaystyle f(3(x-2))\)
  3. \(\displaystyle -3f(x+2)-5\)
  4. \(\displaystyle 3f(-3(x-2))-7\)
  5. \(\displaystyle 5f(3(x+2))+7\)
