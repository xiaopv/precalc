Task 7.16.

Let us find the exact value of the expression \(\tan(\arccos\frac{2}{3})\text{.}\)
  1. First, let us call \(\arccos(\frac{2}{3})=\alpha\text{.}\) Create a right triangle such that one of the non-right triangles is \(\alpha\) and \(\cos(\alpha)=\frac{2}{3}\text{.}\)
  2. Second, find out the value of \(\tan(\alpha)\text{.}\)
  3. Third, find out the value of \(\tan(\arccos\frac{2}{3})\text{.}\)
